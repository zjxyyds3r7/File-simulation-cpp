# 一个模拟文件系统的小demo by周景鑫

1. 所有数据都存放在内存中 整个是树形结构

2. 只支持几个基础操作 因为懒阉割了很多功能

   1. 文件创建 mkfile
   2. 文件夹创建 mkdir
   3. 写入文件 write
   4. 显示文件内容 show
   5. 复制文件 copy
   6. 删除文件 del
   7. 列出目录所有文件 ls
   8. 帮助 help

3. 代码风格非常差 .h与.cpp分离几乎没做 千万别学！！

4. 写了大半天我也不知道有什么问题 但是自己测试这些功能都是能用的耶
